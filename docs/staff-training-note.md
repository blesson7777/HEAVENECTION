# Staff Training Note

## Customer Calling Workflow

### Main purpose
- Call the assigned customer properly
- Mark the correct result after the call
- Keep the work flow moving without leaving any customer unfinished

### Before calling
- Open the next customer from your list
- Read the customer name and phone number carefully
- Make sure you are ready before starting the call

### While calling
- Speak politely and clearly
- Introduce yourself properly
- Listen to the customer before deciding the result
- Complete the customer properly before moving to the next one

### After the call
- Wait for the result screen
- Mark the correct result based on the actual conversation

### Meaning of each result
- `Follow Up`
  Use this when the customer is positive and useful for the next business step.
- `Call Back`
  Use this only when the customer asks to be called later.
  Staff must choose both the callback date and the callback time slot.
- `Rejected`
  Use this only when the customer clearly says they are not interested.
- `No Response`
  Use this when the customer did not attend the call or the discussion did not happen properly.

### Important callback rule
- A callback is a customer-requested later discussion
- It is not the same as a normal positive lead
- Callback customers return at the selected date and time slot as priority calls

### Important work rules
- Do not skip customer result marking
- Do not move to the next customer while the current one is unfinished
- Do not mark the wrong result just to finish fast
- Keep making calls continuously during work time
- If there is a long gap without calling, that gap is not treated as working time

### How work time is valued
- Work time is based on real calling activity
- Continuous genuine calling work is valued
- Empty quick attempts without real calling effort do not help
- Very short invalid calls are not treated as valid work activity

### Good staff practice
- Call sincerely
- Mark the real result
- Use `Follow Up` only for positive leads
- Use `Call Back` only when the customer asks for another time
- Keep the customer flow clean and complete

### Final note
Correct calling and correct result marking are both important.
The system values real calling effort and proper customer handling.
